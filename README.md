# SKALA-FRONT

SKALA 프론트엔드(HTML/CSS/JavaScript) 실습 프로젝트입니다.

## 폴더 구조

파일들이 서로 상대경로(`href`, `src`)로 연결되어 있어 기능별로 나뉜 구조를 그대로 유지했습니다.

- `html/` : 페이지 마크업 (index, myClass, myHoliday, myProfile, myTrip, signup 등)
- `css/` : 스타일시트
- `script/` : 자바스크립트 로직
- `media/` : 이미지 · 영상 · 오디오 리소스
